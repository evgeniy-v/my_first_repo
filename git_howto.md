# Подказка по GIT

## Создание репозитория:
```sh
git init
```

## Добавление изменений:
```sh
git add
```
## Создание коммитов:
```sh
git commit -m "Message"
```

## Просмотреть список коммитов:
```sh
git log
```
## Просмотреть список коммитов в кратком виде:
```sh
git log --oneline
```

## Переход между коммитами
```sh
git checkout
```
## Просмотр состояние репозитория
```sh
git status
```
